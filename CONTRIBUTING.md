# Cómo contribuir

## Licencia de las contribuciones

Este proyecto está licenciado bajo **AGPL-3.0-or-later** (ver [`LICENSE`](./LICENSE)).

Al enviar una contribución (pull request, parche, o cualquier otra forma), aceptás que:

- Tu contribución queda licenciada bajo **AGPL-3.0-or-later**, la misma licencia que el resto del proyecto.
- Conservás el copyright sobre tu propia contribución. No le estás cediendo el copyright a quien mantiene el proyecto, y no hace falta firmar ningún Contributor License Agreement (CLA).

## Por qué importa esto

Como cada contribuyente conserva su propio copyright, ninguna persona por sí sola (ni siquiera el autor original) puede relicenciar este proyecto bajo otros términos, por ejemplo unos propietarios. Relicenciarlo requeriría el acuerdo de todos los titulares de copyright. Esto es intencional: mantiene el código abierto para quien lo use o construya sobre él, incluso como servicio de red, mientras el proyecto exista.

Si no te sentís cómodo licenciando tu contribución de esta forma, por favor no la envíes.
